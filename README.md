MooWoo
======

A distributed programming framework : Master-Worker Pattern


				Master    - Zookeeper
			  /   |    \
			 /          \
			/     |      \
	  Worker1  Worker2  Worker3


Rank Style:
	Master 	- Rank 0
	Worker1 - Rank 1
	...












Dependency Tools & Libraries:
	Poco C++  
	Zookeeper
